Question: - If you create a constructor method to build a DVD by title then create a constructor method to build 
a DVD by category. Does JAVA allow you to do this?
Không. Vì Java phân biệt các constructor dựa trên kiểu dữ liệu của tham số (Parameter types). Cả hai đều là String nên Java sẽ coi chúng là một, dẫn đến lỗi trùng lặp (Compile-time error).
